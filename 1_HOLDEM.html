Hold'Em

Jeu 2D — Python (Arcade) — GameJam ESIEE Paris, équipe de 5

codeVoir sur GitHub
← Retour aux projets

📌 Description

Jeu réalisé en Python avec la librairie Arcade, lors d'une GameJam de deux jours et demi en équipe de 5 à ESIEE Paris. Le thème imposé était « Vous n'êtes pas le centre de l'histoire » : plutôt que d'incarner le héros, le joueur prend le contrôle de ses ennemis et doit chercher à le ralentir le plus possible avant qu'il n'atteigne son objectif.

🗺️ Structure du jeu

Le jeu s'organise autour d'un menu principal, d'une carte permettant de naviguer librement entre les différentes scènes, et de cinématiques (cutscenes) qui ponctuent la progression de l'histoire. Trois scènes jouables proposent chacune une situation différente où le joueur doit ralentir le héros en prenant le contrôle de ses ennemis.

🛠 Technologies utilisées
  • Langage : Python
  • Librairie de jeu 2D : Arcade
🎮 Comment lancer le jeu ?
  1. Cloner le dépôt GitHub
  2. Installer la dépendance : pip install arcade
  3. Lancer main.py
