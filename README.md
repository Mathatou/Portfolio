# Portfolio — AUBRY Mathias

Portfolio personnel présentant mon parcours, mes compétences et mes projets, réalisé en HTML / CSS / JS avec le framework [Materialize CSS](https://materializecss.com).

## 🔗 Liens

- **GitHub** : [Mathatou](https://github.com/Mathatou)
- **LinkedIn** : [mathias-aubry](https://www.linkedin.com/in/mathias-aubry/)

## 🛠 Stack technique

- HTML5 / CSS3
- JavaScript (jQuery)
- [Materialize CSS](https://materializecss.com) (composants, grille, parallax)

## 📂 Structure du projet

```
.
├── index.html              # Page d'accueil (présentation, formation, projets, hobbies)
├── 1_WDTT.html                # Page détaillée — Where Darkness Took Them
├── 1_Color-Shooter.html       # Page détaillée — Head In The Cloud
├── css/
│   ├── materialize.css        # Framework Materialize
│   ├── style.css              # Styles globaux du site
│   ├── wdtt.css                # Thème dédié à la page WDTT (horreur, rouge-bordeaux)
│   └── head-in-the-cloud.css  # Thème dédié à la page Head In The Cloud (VR, bleu-violet)
├── js/
│   ├── materialize.js
│   ├── init.js
│   └── fadeIn.js
├── img/                        # Images du site (logo, icônes de projets, etc.)
├── background1.jpg / background2.jpg / background3.jpg   # Images de fond (parallax)
└── CV_AUBRYM.pdf              # CV téléchargeable depuis la bannière d'accueil
```

## ✨ Contenu

- **Accueil** : présentation, parcours (ESIEE Paris / KNDS France Training), aperçu des projets, hobbies.
- **Pages projets** : chaque projet dispose de sa propre page détaillée (description, technologies, assets, équipe, licence) avec une charte graphique qui lui est propre.

## 🚀 Lancer en local

Aucune dépendance ni build nécessaire : il s'agit d'un site statique.

```bash
git clone https://github.com/Mathatou/<nom-du-repo>.git
cd <nom-du-repo>
```

Puis ouvrir `index.html` dans un navigateur, ou servir le dossier avec un petit serveur local, par exemple :

```bash
python3 -m http.server
```
