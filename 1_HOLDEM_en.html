📌 Description

Game developed in Python using the Arcade library, during a 2.5-day GameJam with a team of 5 at ESIEE Paris. The assigned theme was "You are not the center of the story": instead of playing as the hero, the player takes control of their enemies and must try to slow the hero down as much as possible before they reach their goal.

🗺️ Game Structure

The game revolves around a main menu, allowing you to launch the experience. The story then unfolds through cinematic cutscenes. Three playable scenes each feature a different monster controlled by the player, whose objective is to delay the hero.

🛠 Technologies Used
  • Language: Python
  • 2D Game Library: Arcade
🎮 How to Launch the Game?
  1. Clone the GitHub repository
  2. Install the dependency: pip install arcade
  3. Run main.py
