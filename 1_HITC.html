📌 Description du projet

Head In The Cloud est un jeu en réalité virtuelle réalisé dans le cadre de l'unité Projet Multidisciplinaire 4 à ESIEE Paris en E4FI. Il s'agit d'une expérience d'exploration, de tir et de réflexion. Le joueur commence dans un lobby central (forêt avec feu de camp) et traverse des portails pour accéder à trois expériences distinctes : un mini-jeu de tir dans un univers Far West, un labyrinthe d'horreur, et un escape-game de réflexion.

🛠 Technologies utilisées
  • Moteur de jeu : Unity
  • Langage de programmation : C#
  • Logiciels tiers : GitHub, GIMP, Blender, Audacity
  • Casque VR : Meta Quest 2
🎮 Comment jouer ?
  1. Télécharger sur GitHub la release du jeu
  2. Récupérer l'APK
  3. Transférer l'APK sur le casque
✨ Points forts techniques
  • Interactivité (XRI) : briquet pour allumer le feu de camp, portails de téléportation, tir au pistolet via raycasting avec scoreboard dynamique, collecte d'orbes, manipulation de tiroirs, manivelles et échelles dans l'escape game avec énigmes randomisées.
  • Confort VR : deux modes de déplacement (continu au joystick gauche, téléportation au joystick droit) pour limiter le motion sickness ; capacité de saut retirée pour la même raison.
  • Optimisation Quest 2 : direction artistique low-poly, brouillard du labyrinthe géré par un système de particules plutôt qu'une brume globale, shaders personnalisés via ShaderGraph.
  • Rendu : shaders sur-mesure pour les portails (distorsion, couleur unique par niveau), le feu de camp, l'oscillation d'objets guidant le regard, des decals, ainsi que des effets VFX (muzzle flash, explosions).
  • Immersion : scènes à taille humaine, ambiance sonore propre à chaque univers (musique et bruitages en partie enregistrés par l'équipe), onboarding progressif dans le lobby.
🎲 Inspirations
  • VRChat → le hub central reliant les niveaux par des portails
  • Beat Saber → le mini-jeu de tir sur cibles de couleurs
  • Where Darkness Took Them → le labyrinthe d'horreur, amplifié par le passage en VR
