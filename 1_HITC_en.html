📌 Project Description

Head In The Cloud is a virtual reality game developed as part of the Multidisciplinary Project 4 course at ESIEE Paris in E4FI. It offers an experience combining exploration, shooting, and puzzle-solving. The player starts in a central lobby (a forest with a campfire) and steps through portals to access three distinct experiences: a shooting mini-game set in a Wild West universe, a horror labyrinth, and a puzzle-driven escape game.

🛠 Technologies Used
  • Game Engine: Unity
  • Programming Language: C#
  • Third-party Software: GitHub, GIMP, Blender, Audacity
  • VR Headset: Meta Quest 2
🎮 How to Play?
  1. Download the game release from GitHub
  2. Get the APK file
  3. Transfer the APK to your headset
✨ Technical Highlights
  • Interactivity (XRI): use a lighter to ignite the campfire, teleportation portals, pistol shooting via raycasting with a dynamic scoreboard, orb collection, interacting with drawers, handcranks, and ladders in the escape game featuring randomized puzzles. Decals are used to display clues for solving the puzzles.
  • VR Comfort: two locomotion modes (continuous movement via the left joystick, teleportation via the right joystick) to reduce motion sickness; jumping capabilities were removed for the same reason.
  • Quest 2 Optimization: low-poly art direction, labyrinth fog managed by a particle system rather than global fog, custom shaders built with ShaderGraph.
  • Rendering: tailor-made shaders for portals (distortion effect, unique color per level), the campfire, object oscillation to guide the player's gaze, decals, and VFX (muzzle flashes, explosions).
  • Immersion: human-scale scenes, custom soundscapes for each universe (music and sound effects partially recorded by the team).
🎲 Inspirations
